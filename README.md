Калькулятор-строка - быстрый компактный помощник по проведению расчётов.
Он укажет на ошибки, проведёт высокоточный расчёт, обеспечит быстрый ввод и умное редактирование.
У проекта стильный вид, нет вспомогательного графического интерфейса, зато знание огромного множества горячих клавиш даёт огромный прирост к скорости работы
