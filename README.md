Калькулятор-строка - быстрый компактный помощник по расчётам.
Укажет на ошибки, проведёт высокоточный расчёт, обеспечит быстрый ввод и умное редактирование.
У проекта стильный вид, нет "кнопочного" графического интерфейса, зато знание его горячих клавиш даёт заметный прирост к скорости твоей работы
